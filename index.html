📖 Logique de calcul — conforme Excel SAPET
Toutes les formules reproduites à l'identique depuis le fichier SUIVI_POMPAGE.xlsx — SAPET Energy.
Champs SAISIR (vert)
• Hauteur liquide avant/après (mm)
• Température LIQUIDE avant/après (°C) → T°liq utilisée pour VCF
• Température GAZ avant/après (°C) → T°gaz utilisée pour poids spéc. air
• Densité à 15°C avant (kg/L) → saisie directe
• Pression relative avant/après (bar)
• Densités navire vers S01/S02/S03
Formules AUTO (bleu) — identiques Excel
H_gaz = 24803 − H_liq (mm) V_liq = VLOOKUP(H_liq, barème_Sx) / 1000 (m³) V_gaz = VolTot_Sx − V_liq (m³)
E = AC17/AC18 = 58.12/29 = 2.004137931 G = Pression_relative + 1 (bar absolu) H = ROUND(0.35318295/(273.15 + T°_GAZ), 6) × 1000 (kg/m³)
— Arrondi densité pour VCF (Q10→Q14) : D×100 → frac = D100 − TRUNC(D100) D_arr×100 = TRUNC(D100) + {0 si frac≤0.245 | 0.5 si ≤0.745 | 1} D_arr = D_arr×100 / 100 — Arrondi température pour VCF (Q16→Q18) : frac_T = T − TRUNC(T) T_arr = TRUNC(T) + {0 si frac≤0.245 | 0.5 si ≤0.745 | 1} — Lookup VCF (Q20) : VCF = INDEX(Table54E, MATCH(T_arr, col_temp, 0), MATCH(D_arr, row_dens, 1))
D_temp = ROUND(D15 × VCF, 4) Poids_liq = V_liq × D_temp (T) Poids_gaz = (V_gaz × E × H × G) / 1000 (T) Poids_total = Poids_liq + Poids_gaz (T)
D15_après = ((D15_av × V_liq_av) + (ρ_navire × V_reçu)) / (V_reçu + V_liq_av) V_reçu = V_liq_après − V_liq_avant (m³)
VOLUME REÇU = V_liq_après − V_liq_avant (m³) TONNAGE REÇU = Poids_total_après − Poids_total_avant (T)
